To use https://cloudevents.io[CloudEvents], add the following dependency:

dependency:micronaut-gcp-function-cloudevents[groupId=io.micronaut.gcp]

In your application, create a class which extends api:gcp.function.cloudevents.GoogleCloudEventsFunction[]. Google publishes POJOs for https://github.com/googleapis/google-cloudevents-java[Google Cloud Events].
